# RCuber
Rust Cuber, 一个Rust版本的魔方(Rubick's Cube)库，支持终端展示和CFOP解法。

### 基本设计
1. [kociemba](https://crates.io/crates/kociemba)的基础上，实现CFOP算法。

### Crates.io
* https://crates.io/crates/rcuber
